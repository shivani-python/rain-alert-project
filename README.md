# Rain Alert Project

## 📌 Description
This project sends email alerts when rain is expected using weather API data.

## 🚀 Features
- Fetches weather data using API
- Checks rain conditions
- Sends email notification automatically

## 🛠️ Technologies Used
- Python
- Requests
- SMTP
- API (OpenWeatherMap)

## ▶️ How to Run
1. Install required libraries:
   pip install requests

2. Run the file:
   python main.py

## 📧 Output
Sends an email alert if rain is predicted.
